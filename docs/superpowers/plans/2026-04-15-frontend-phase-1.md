# Frontend Phase 1 — Foundations Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Take the approved `PortfolioSummary.vue` mockup from "fixture-backed demo" to "production-ready foundation" — typed, linted, tested, mocked, auth-wrapped, and fetching real data from a contract-based mock backend.

**Architecture:** Vue 3 `<script setup>` + TypeScript with strict mode. Pinia stores own UI state (theme, scroll). TanStack Vue Query owns server state (fetching/caching/retries), backed by an `ofetch` client whose types are codegen'd from `api/openapi.yaml` via `openapi-typescript`. Auth0 SPA SDK handles login + bearer-token attachment. MSW intercepts API calls in dev/test with handlers matching the OpenAPI spec. Theme toggle persists to localStorage; first-visit default comes from OS `prefers-color-scheme`.

**Tech Stack:** Vue 3.5, Vite 6, TypeScript 5.7, Pinia 2.3, @tanstack/vue-query 5, ofetch 1, openapi-typescript 7, @auth0/auth0-vue 2, msw 2, Vitest 2, @vue/test-utils 2, Playwright 1.49, ESLint 9 flat config + eslint-plugin-vue + @typescript-eslint, Prettier 3.

```ts
import { defineStore } from 'pinia'
import { ref, watch } from 'vue'

const THEME_KEY = 'pv:theme'

type Theme = 'light' | 'dark'

function readStoredTheme(): Theme | null {
  if (typeof window === 'undefined') return null
  const raw = window.localStorage.getItem(THEME_KEY)
  return raw === 'light' || raw === 'dark' ? raw : null
}

function systemTheme(): Theme {
  if (typeof window === 'undefined' || !window.matchMedia) return 'dark'
  return window.matchMedia('(prefers-color-scheme: light)').matches ? 'light' : 'dark'
}

export const useUiStore = defineStore('ui', () => {
  const theme = ref<Theme>(readStoredTheme() ?? systemTheme())

  watch(theme, (value) => {
    if (typeof window !== 'undefined') {
      window.localStorage.setItem(THEME_KEY, value)
    }
  })

  function toggleTheme(): void {
    theme.value = theme.value === 'light' ? 'dark' : 'light'
  }

  function setTheme(next: Theme): void {
    theme.value = next
  }

  return { theme, toggleTheme, setTheme }
})
```
